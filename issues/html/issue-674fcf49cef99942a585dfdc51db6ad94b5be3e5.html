Graph Unrolling Tool

« CFGAnalysis project page

PVF analysis needs a single instruction stream, so unroll the CFG.

Syntax: ./unroll -f file.cfg -b 1,2,3,2,3,4

with -f -> CFG file -b -> comma-separated list of BBs (tool asserts they are indeed connected!)

Later, dynamic analysis can be used to generate the BB list.
